### 协议说明与示例

#### 说明：
1. msgid根据public.h里面定义的枚举来设定
2. 嵌套的时候使用对象形式，不得使用字符串化的json
3. data中一定要有内容，否则验证不通过。其他根据返回消息可以得到。

#### 1、以下为密码登录请求

```json
{
    "msgid": 1000,
    "timestamp":1633024567890,
	"version": "1.0",
	"token":"",
    "data":{
		"device":"pc",
		"userid":123,
        "password":"加密密码"
    }
}
```

#### 响应
```json
{
  "msgid": 1001,
  "code": 0,
  "message": "OK",
  "timestamp":1633024567900,
  "data": {
    "friends": [
    	{"userid": 1,"username": "lisi1","state": "online"},
		{"userid": 2,"username": "lisi2","state": "online"}
    ],
    "groups": [
		{"groupid":"1","groupname":"bb","groupdesc":"cc"},
		{"groupid":"2","groupname":"bb","groupdesc":"cc"}
	],
	"token":"xxxxx"
  }
}
```

#### 2、以下为私聊请求

```json
{
    "msgid": 4000,
    "timestamp":1633024567890,
	"version": "1.0",
	"token":"xxxxx",
    "data":{
		"toid":1,
		"msg":"hello world"
    }
}
```
